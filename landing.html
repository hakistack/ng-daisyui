Forms

Dynamic Form

Build complex forms declaratively with a type-safe builder API. Supports 24 field types, conditional logic, wizard stepper, auto-save, and responsive grid layouts.

24 Field Types

text email password number tel url textarea select multiselect radio checkbox toggle date datetime-local time range file color hidden

Layout Modes

vertical horizontal grid (1-12 cols)

Responsive grid with per-field colSpan supporting breakpoints: default, sm, md, lg, xl, 2xl. Field width options: full, 1/2, 1/3, 1/4, 2/3, 3/4, auto.

Conditional Logic

showWhen hideWhen requiredWhen disabledWhen

Operators: equals, not-equals, contains, greater-than, less-than, in, not-in, function. Supports static values or dynamic callback functions.

Wizard Stepper

linear mode step validation step summary review step

Multi-step forms with step.create() and step.review(). Custom button labels per step, optional steps, and animated transitions.

Auto-Save

localStorage API endpoint debounced clear on submit

Configured via provideFormState(). Automatically saves and restores form state with FormStateService.

Validation

required minLength / maxLength min / max pattern (regex) email custom ValidatorFn

Dynamic Options

optionsFrom field async loadFn Promise / Observable loading placeholder

Load select/radio options dynamically based on another field's value. Supports sync arrays, Promises, and Observables.

// Type-safe form with grid layout, wizard, and conditional fields
const form = createForm({
  steps: [
    step.create('Personal', [
      field.text('firstName', 'First Name', { required: true, colSpan: { default: 12, md: 6 } }),
      field.text('lastName', 'Last Name', { required: true, colSpan: { default: 12, md: 6 } }),
      field.email('email', 'Email'),
    ]),
    step.create('Role', [
      field.select('role', 'Role', { choices: ['Admin', 'Editor', 'Viewer'] }),
      field.toggle('isManager', 'Is Manager?'),
      field.number('teamSize', 'Team Size', {
        showWhen: ['isManager', true],  // only visible when isManager is true
        min: 1, max: 100,
      }),
    ]),
    step.review('Review'),
  ],
  layout: 'grid', gridColumns: 12,
  autoSave: { enabled: true, formId: 'onboarding', debounceMs: 1000 },
  onSubmit: (data) => api.createUser(data.values),
});
// Template: <hk-dynamic-form [config]="form.config()" /> <button (click)="form.submit()">Submit</button>

Infrastructure

Services & Providers

Tree-shakeable services configured via Angular's dependency injection. Everything is optional.

Provider Service Purpose
provideFormState() FormStateService Auto-save/restore form state. Modes: localStorage (custom key prefix) or api (POST/GET to endpoint)
providePipes() PipeRegistryService Apply pipes programmatically. Built-in: currency, date, decimal, json, keyValue, lowercase, percent, titlecase, uppercase. Supports custom pipes via CUSTOM_PIPES token
provideAlert() AlertService Promise-based modal dialogs. i18n via translate function and langChange$ observable
provideToast() ToastService Notification stacking. Global config: maxToasts, preventDuplicates, extendedTimeOut, exitDuration
provideHkTheme() ThemeService Theme compatibility layer for DaisyUI v4 and v5
provideIcons() IconService Register additional Lucide icons beyond the 29 built-in defaults
DialogService DialogService Angular CDK Dialog wrapper with component injection and route-aware auto-close
Helpers

Pagination Helpers

Signal-based pagination state management for both cursor and offset strategies. Generates HttpParams for your API automatically.

Cursor createCursorPagination() 

const pg = createCursorPagination({ pageSize: 20 });

pg.canGoNext    // Signal<boolean>
pg.canGoPrev    // Signal<boolean>
pg.isLoading    // Signal<boolean>
pg.goNext()      // returns next cursor
pg.goPrev()      // returns prev cursor
pg.getHttpParams() // HttpParams ready for API
pg.updateFromResponse(apiResponse)

Offset createOffsetPagination() 

const pg = createOffsetPagination({
  pageSize: 25, totalItems: 500,
});

pg.totalPages   // Signal<number> (computed)
pg.offset       // Signal<number> (computed)
pg.goToPage(3)
pg.goFirst() / pg.goLast()
pg.changePageSize(50)
pg.getHttpParams() // limit, offset, sort

Get Started

Up and Running in Minutes

1

Configure the registry

Add to your project's .npmrc:

@hakistack:registry=https://hakistack-registry.fly.dev
2

Install

npm install @hakistack/ng-daisyui
3

Import and use

import { DynamicFormComponent, createForm, field }
  from '@hakistack/ng-daisyui';

@Component({
  imports: [DynamicFormComponent],
  template: `
    <hk-dynamic-form [config]="form.config()" />
    <button (click)="form.submit()">Submit</button>
  `,
})
export class MyComponent {
  form = createForm({
    fields: [
      field.text('name', 'Name', { required: true }),
      field.email('email', 'Email'),
      field.select('role', 'Role', {
        choices: ['Admin', 'Editor', 'Viewer'],
      }),
    ],
    onSubmit: (data) => console.log(data),
  });
}
